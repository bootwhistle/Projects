Assignment 1

By this time after taking the course for two weeks, you should have learned enough about React to create your first React App with a simple component.

Learning Outcome

  1. Build reusable components in React.

Scenario

You have joined a new team of developers that are currently working on building a simple weather and temperature app that can help the users to see latest weather updates. The requirements of the app will be as following:

  1. The application should display the current weather and temperature for a specific location.
  2. The user should be able to search for a location and see the corresponding weather information.

Directions

In order to complete the task, you will have to go through the following list of sub-tasks:

  • Download the provided stylesheets and JSON file for this assignment.
  • Create an empty React application.
  • Create the proper folder structure and components required to build the final list component (i.e. /components/WeatherDisplay).
  • Write the components code that will first display a single static item.
  • Pass to the Weather App a static list of locations, provided in this assignment, from the root APP component and the component will dynamically loop over this list and display it on the home page.

Notes

  • Please use the stylesheets provided in this assignment for each component.
  • During this assignment the app will not be connected to an API for live date updates, and instead there will be a JSON data sample provided in a separate file to use as the data source.

Submission Guidelines

Submit the link to your GitHub repository to the Assignment 1 Dropbox on D2L.
